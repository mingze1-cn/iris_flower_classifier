# 鸢尾花分类系统 - Iris Flower Classification System

基于多种机器学习算法的鸢尾花分类系统，比较不同模型在经典数据集上的性能。

## 功能特性
- **多模型比较**: SVM、随机森林、逻辑回归
- **自动调优**: 使用GridSearchCV进行超参数优化
- **完整评估**: 准确率、分类报告、混淆矩阵
- **可视化**: Seaborn热力图展示混淆矩阵
- **模型持久化**: 保存最佳模型供后续使用

## 数据集信息
- **数据集**: Iris (鸢尾花)
- **样本数**: 150
- **特征数**: 4 (花萼长度、花萼宽度、花瓣长度、花瓣宽度)
- **类别数**: 3 (Setosa, Versicolour, Virginica)
- **类别分布**: 每个类别50个样本
